«Ad kkseɣ tudmawit deg ufraq n yidellilen»

Deffir n usɣimi n mass Ounoughi Larbi d amasay n ANEP, 
aneɣlaf  n  taɣwalt  mass  Amar  Belhimer  iweɛɛed  merra 
imḍebren  yellan  deg  uḥric  n  taɣwalt,  am  yinemhalen  n 
yiɣmisen inagduden neɣ usligen belli ad iɛawed tamuɣli 
ɣef ufraq n yidellilen akken iwata lḥal. Yerna ad yekkes 
tudmet  deg  uḥric-a  n  yidellilen,  acku  yes-sen  i  teddun 
wallalen-a  n  taɣwalt.  Mebla  idellilen  aɣmis  ad  yenger. 
Ugur  n  ufraq  n  yidellilen  yeqqim  d  tamukrist  deg  uɣlif 
n taɣwalt.

Tamsalt-a n ufraq n yidellilen tecɣeb aṭas annar n taɣwalt. 
Acku  s  yisebtar  n  yidellilen  ara  d-teckem  tedrimt  ɣer 
yimḍebren  n  teẓrigin.  S  tedrimt-nni  i  yezmer  ad  iselḥu 
lecɣal  n  uɣmis  neɣ  n  tiliẓri.  Am  lexlas  n  yineɣmasen, 
asuffeɣ  n  tebzert,  lexlas  n  yixxamen  n  teẓrigin,  wid 
iferrqen aɣmis etg. Aɣmis iwakken ad yelḥu ɣer sdat, ilaq 
ad yesɛu xersum yiwen n usebter n yidellilen.

Aseggas-a 2020, Tuddsa tamaḍalt n 

tezmert textar tamsujjit d tazzayrit 
iwumi ara d-terr tajmilt s yisem-is, 
i  temjujiyin  d  lqablat    akk  yellan  di 
Tferka.  Tamsujayt-a  tazzayrit,  isem-is 
Karima  Ɛazzug,  tesɛa  25  d  aseggas  deg 
uxeddim-a  deg  sbitar  El-Kettar,  di  Lez-
zayer-Tamaneɣt.
Deg temlilit-a, Tuddsa tamaḍalt n tezmert 
isuli-d  tadiwennit  d  Karima  Ɛazzug,  i 
d-yesres deg site-ines, anda i d-tenna ɣef 
lxetyar  n  uxeddim-a  :  «  Asmi  d-yewweḍ 
lweqt  ad  xtireɣ  abrid  ara  ḍefreɣ  deg  tu-
dert-iw,  uɣeɣ  win  n  tezmert  anda  i  ufiɣ 
iman-iw,  acku  ḥemleɣ  ad  ɛineɣ  wiyiḍ,  i 
liɛana n medden, ulac acu yifen axeddim 
n  temsujayt.  D  axeddim  ur  ttbeddileɣ, 

ayen yebɣun yili. »
Akken dɣa i d-terna unuɣ yebha i d-tezḍa 
s  wawal-is  :  «  Ul-nneɣ  yekkat  d  win  n 
umuḍin.  Ma  yeḥla  nestaḥla.  Nzeddi  i 
uxeddim, anda tal ta, yal wa deg umḍiq-is 
s  uxeddim-is.  Mi  d-imeslay  yiwen  ɣef 
lemḥadra n umuḍin, aya ur yeɛni yara ad 
s-yettakk  yiwen  ddwa.  Ilaq  ad  imended 
yiwen ɣef waṭṭan i t-yuɣen, ad isikked ma 
yestaḥla, ma itett akken i d-yewwi, zeddi-
git neɣ lufḍan, afra-ines ma yelha neɣ ye-
jmeɛ kulci zdaxel-is… »
Karima Ɛazzug seg tid isebruruzen lecɣal, 
teṭṭafartilufa  alamma  tewweḍ  ar  tagga-
ra-nsent. Akken  daɣ  i  d-tella  d  tin  ixed-
men  tagnit  tugar  tid  n  yimeddukal-is  d 
temdduakl-is n uxeddim, da deg sbitar n 

El-Kettar. « Mi d-usan imsujayen d imay-
nuten  i  usileɣ,  ttiliɣ  deffir-sen  akken  ad 
ten-swilhen ɣer wayen ur ssinen d wayen 
ilaqen, akken ad ilin seg wid yettwillihen 
i  uxeddim-nsen  acku  gar  yifassen-nsen 
tres terwiḥt, ad issinen ad slen i umuḍin, 
ad  rnun  ad  t-lawin  ».  i  d-tenna  deg 
tdiwennit  i  s-texdem  Tuddsa  tamaḍalt  n 
tezmert.
Karima  Ɛazzug  tefreḥ  s  uxeddim-is  :  « 
Ttḥulfaɣ s tayri deg wul, mi mlaleɣ d yi-
muḍan. Mi d-jban, gellun-d s liḥal n win 
yeṭṭfen  tarwiḥt  deg  uẓar  rqiq,  mi  ḥlan 
ffɣen udem inecraḥ, ferḥeɣ s wayen xed-

ddunit, yerna tɛaneḍ-t, acu ara ternuḍ ? »
