Γas, aṭas n txazabiyin i d-yeɣlin ɣef tmurt, ta d asubbu n
ssuma n tbettit n lpitrul iḍurren nezzeh tadamsa n tmurt, ta
d anfafad coronavirus i yɛawden tikli i umaḍal… maca, kra

d-tuɣ yakan, ta tessarem ad tbeddel tikli. Lezzayer, gar tid
yebɣan ad d-tbeddel tikli, ad tegzem aseqqen d yir lecɣal
ansa i d-tekka. Gar ubeddel i d-iheyya uselway n Tigduda,
yella win uɛiwed n tmuɣli ɣef yilugan n Tmedawt. Ladɣa,
aya  d  ayen  i  d-yenna  asmi  d-ibedd  ɣef  uqerru  n  tmurt.

wid ilaqen, akken ar taggara, ad skasin fell-as. Asenfar-a
amaynut n Tmendawt, yettwabeddlen i d-theyya tesqamut
n  yimassanen  n  uzref,  iɣef  Ḥmed  Laraba  i  d-yesbedd
uselway  n  Tigduda  ɣef  uqerruy-is,  i  d-ibedden  i  cɣel-a,
win n uqeddem n tmuɣliwin yizerfanin, ara yaɣ amur, d
amagrad deg Tmendawt-a tamaynut sway ara teddu tmurt.

d leqder i sɛant tezyiwin-is deg tmura n tugdudt. Deg tmurt-nneɣ,
tamendawt tecba “akustim” : yal m’ara d-yali uselway ajdid, ad
d-ixiḍ yiwet ara yeffɣen fell-as. Yuɣal uḍris n tmendawt tazzayrit
amzun d aḍris n tsekla, yal aselway yessenfay-d deg-s tamuɣli-s
d lebɣi-s mebla ma tedda-d tmuɣli d lebɣi n uɣref azzayri. Neḥḍer
mi yettwakkes wazal d leqder i tmendawt, nwala amek tt-ttawin
yiselwayen deg ubrid n leḥkem d aɛwin-nsen. Seg 1962 ar ass-a,
ulac aselway ur d-newwi tamendawt-“is”. Aneggaru, d Teboune
i d-ilemden amek i ttbeddilen tamendawt, imi yeḥḍer i wid i
t-yezwaren asmi i ttxiḍin “ikustimen-nsen”. Atan yessuffeɣ-itt-
id i usqerdec. Ayen i d-ibanen akka kan sufella, d aḍris icebḥen,
yerqem s tenfaliyin ḥlawen. D aḍris yeččuren d tilelli d yizerfan n
umdan. D aḍris i yessawalen i ureṣṣi n tugdudt d lebni n uwanek
atrar. Acu kan, ayen ur ilaq ad t-nettu, d ayen iɛeddan fell-aɣ : ula
d tamendawt n Bouteflika ur tt-ixuṣ wara. Ihi, ur teqqim ara deg
uḍris n tmendawt ma yecbeḥ neɣ yerqem ma mačči d aɣref i t-id-
yuran. Ur teqqim ara ma yeččur d tilelli neɣ d tugdudt ma yettidir
ugdud ala leḥris d tugdi deg tudert-is. Ur teqqim ara deg tmendawt
ma tbeddel neɣ ala, imi leqwanen i d-sufuɣen seg-s wid iseddayen
tilufa n tmurt, d yir leqwanen : d leqwanen yeddan dima ɣer tama n
yimḍebren, d awezɣi ɣer tama n ucaɛbi.
D leqwanen isekcamen leḥbus, mačči d wid iḍemnen tilelli n
umeslay wala n rray. Yal aselway s tmendawt-is, ma d tamendawt n
uɣref, deg ubrid i tt-yura.

iw,  s  wawal  ifaw,  usrid:  Yuklal
Yidir  asgunfu  n  lebda  deg  wakal  n
baba-s  ameksa  bu  tissas!  Imeqqranen
akk,  llan  d  imeksawen  neɣ  qrib  i
t-uɣalen. Ɛisa, azul fell-as d aḥmayan,
d  yiwen  seg-sen,  yella  d  ameksa.  D
aɣilif  deg  unezgum  ;  tamezwarut  ɣef
teqriḥt-a i d-yeɣlin fell-aneɣ am lebreq
; iruḥ-aɣ, ur tuḥtam ucennay ur nesɛi
tamtilt, alsawi uslig, tis snat d amuqel
deg  wakal-niḍen  yexḍan  win  n
yimarawen-is, ad iger irebbi i tfekka n
Yidir, i lebda.
D  tidet,  Yidir,  d  acennay  agraɣlan,  d
aɣerman  n  umaḍal,  maca  amaḍal-a
ameqqran, yesɛa ansi i ibeddu, anda i
ifuk d tlemmast. D taddart n At Leḥsen
kan deg Wat Yanni i izemren ad tili d
beddu d tlemmast-a. Qudreɣ s wannuz
taɣtast  n  twacult  n  Yidir  ɣef  ufran  n
wadeg  n  temḍelt.  Maca,  walaɣ  belli
Yidir,  yuklal  ad  yesɛu  akal  n
yimarawen-is d asummet n lebda.
Ad slen i uẓawan-is, ad t-ǧǧen ad isel i
tidak-nsen.  Akal  n  tmurt  n  Leqbayel,
yessen amek i yettmeslay i warraw-is
ufrinen,  ama  mmuten,  ama  ddren,  d
iminigen  neɣ  ttidiren  da,  qerben  neɣ
beɛden.  Du  leqrar,  ad  uɣalen  akk  s
irebbi-s.  Lmut,  d  tamyiyt,  akka  i
ttwaliɣ  Dda  Ḥamid  neɣ  Idir,  yeɣreq
deg telwit tarebbanit, tiṭ-is ɣef udrar n
Ǧerǧer. Yidir, d Ǧerǧer-nneɣ wis sin.
D tidet, tewwi-d ad tettwaqader teɣtast
n twacult ; afran n temḍelt anda-niḍen
yettwaqader, meɛna, ḥulfaɣ i teqreɣ i
yeqqazen s telqey deg-i, mi ara waliɣ
tamurt  n  Leqbayel  d  Lezzayer,
ttwaḥerrment seg uẓekka n Yidir. Ula
d aẓekka, d tamurt! Si temẓi-s, tardast-
nni  ines,  tettwaheyya-as  ɣer  tama  n
uselmad-is  Dda  Lmulud  neɣ  Mulud
Mɛemmri,  yal  yiwen  amek  i  as-
yessawal.
Win  i  as-yeslemden  ilugan  imezwura
n  tutlayt  n  tmaziɣt  deg  tesdawit  n
Lezzayer  tamanaɣt.  Amkan-is,  ɣer
tama n uɛessas-a n At Yanni. Amkan-
is  n  lebda,  akka  ay  t-ttwaliɣ,  ur
yettibɛid  ara  aṭas  ɣef  uẓekka  n
umussnaw  Ayt  Ḥmed  i  yeṭṭsen  deg
taddart-is  Ɛin  Lḥemman.  Amkan  n
lebda  n  Yidir,  ttwaliɣ-t  ɣer  tama  n
umedyaz  Ṭaher  Ǧaɛut  i  yeɣlin  s
iɛdawen  n
terṣaṣin  n  at
umgarad  d  uberrar,  Ṭaher  Ǧaɛut  i
yeṭṭsen deg Ulxu deg tɣaltin n Uẓeffun.
Amkan-is  n  lebda,  atan  ɣer  tama  n
gma-s
iken  Meɛtub  Lwennas,
aɣewwaɣ  i  iruḥen  d  ameẓyan,  ula  d
netta s terṣaṣin timeksanin n yiɛdawen
n tudert, Lwennas i yeṭṭsen di Tewrirt
Musa  deg  Wat  Dwala.  Iḍes-ines  n
lebda, ttwaliɣ-t, ur yebɛid ara ɣef Crif
Xeddam yettwameḍlen di Bu Mesɛud
deg  Yiferḥunen.  Ɣer  tama  kan  n
Muḥemmed Belḥanafi i yeṭṭsen di Sidi
ičumar,
