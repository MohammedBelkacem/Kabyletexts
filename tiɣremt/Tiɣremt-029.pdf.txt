Mehdi Zeffane yerra-as tajmilt 
i usleɣmay aɣelnaw Ǧamal 
Belmaḍi i as-yefkan awellih-a 
n wurar deg telɣuɣa n tmurt 
n Rrus seld mi as-tserreḥ 
terbaɛt n Rennes. Zeffane i 
yewwin taqbuct n tefriqt akked 
terbaɛt taɣelnwat, iban-d ɣer 
ṣṣenf n yimeqqranen i tikkelt 
tamezwarut deg useggas n 
sin wagimen d mraw d sin, 
ɣer terbaɛt n Liyu, yurar snat 
n tmerwin d ṣṣa n temliliyin 
teɣzi n kraḍ n yiseggasen. 
Deg useggas n sin wagimen 
d mraw d semmus yuɣal ɣer 
terbaɛt n unnar n Rennes 
SRFC, yurar sḍis n tmerwin d 
sḍis n temliliyin teɣzi n ukkuẓ 
n yiseggasen, ɣas akken d 
amḥaddi yewwi-d sin n yiswiyen 
d terbaɛt-a. Leɛqed-is akked 
terbaɛt n SRFC yekfa deg 
wayyur-a n yennayer yezrin, 
imḍebbren n terbaɛt ugin ad 
as-d-rnun leɛqed-nniḍen, serḥen-
as. Iwelleh-it usleɣmay aɣelnaw 
Ǧamal Belmaḍi ad yefren yiwet 
deg terbaɛt n tmurt n Rrus i 
yebɣan ad t-yawin. D ayen 
i yexdem umyurar-a ifernen 
tarbaɛt n Krylia Sovetov Samara 
i yetturaren deg uḥric amenzu 
n telɣuɣa, Mehdi yufa iman-
is deg telɣuɣa n tmurt n Rrus 
syin yessenmer aṭas asleɣmay 
n terbaɛt taɣelnawt i t-iwelhen 
ɣer rray-a. Deg tdiwennit-is d 
uɣmis “Foot mercato” n tmurt n 
Fransa, yebgen-d tumert-is imi 
d-yenna: “Ferḥeɣ aṭas imi lliɣ 
gar n yemyuraren n terbaɛt-a, 
seld n sḍis n wayyuren war 
urar ɣer terbaɛt-iw taqburt, tura 
uɣaleɣ-d ar yinurar d ayen iyi-
d-yesferḥen aṭas”. Yerna-d deg 
wawal-is: “Tura sɛiɣ tagnit ad 
uɣaleɣ ɣer uswir-iw i iṣubben 
sḍis n wayyuren-a yezrin, 
ttawilat akk gerrzen iwakken 
ad t-id-saliɣ da deg telɣuɣa 
n tmurt n Rrus, imyuraren d 
yemḍebbren n terbaɛt n KSS 
bedden-d akk ɣer yidis-iw”.

terbaɛt taɣelnawt: “Mi ara
